About Me

My name is Linda, my last name is Moran, and my maiden name is Driver. Back in the eighties, I had a stint as a computer programmer with a large multinational corporation, but didn't like it much, and switched to testing.

I left the company in 1992 on maternity leave, and never returned. Now it's 2019, and I'm self-bootcamping. Trying to learn, all at the same time:

  • Python 3
  • OOP
  • Django
  • Git and Github
  • CSS (minimally)
  • HTML (minimally
  • Javascript (as little as possible)
  • Databases (using Django models)
  • Pycharm
  • Web Scraping
  • External API's
  • Literacy in other IDE's and other Editors such as Sublime

It's a jungle out there. Not easy to find answers. And when I do construct answers, I find it best to take notes and write procedures. Right now, those notes are in a Blogger Blog titled Full Stack Safari, but my hope is to move them here to this web app I am currently building with Python and Django.
